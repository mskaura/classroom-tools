# Math Translations Explorer

**Grade 8 Ontario Curriculum** — Expectations E1.4 and C1.2

## What it does

A single-page interactive tool with three tabs:

### 1. Geometric Translations (E1.4)
- Visual Cartesian plane with grid
- Students set a translation rule (x shift, y shift)
- A polygon moves in real time showing the original and translated image
- Coordinates table updates automatically with A/A' labeling
- "Random Shape" button cycles through different polygons

### 2. Linear Patterns (C1.2)
- Students set rate of change (m) and starting value (b)
- Table of values, algebraic equation, and graph all update in sync
- Labels growing vs. shrinking vs. constant patterns

### 3. Practice Mode
- Randomly generated questions for both geometric and linear translations
- Immediate feedback with correct answer shown on wrong answers
- Score and streak tracker
- Enter key submits answers
